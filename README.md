# CV Analyser – Potential Quest

A free AI-powered CV analysis tool for the German job market.
Built for SAP and AI/Data Engineering candidates.

🔗 **Live tool:** https://potentialquest.github.io/cv-analyser/

## What it does
- ATS Score (0–100) with pillar breakdown
- German CV format checklist
- Keyword gap analysis for SAP and AI roles
- Strengths and priority fixes
- EN/DE language toggle

## Tech
- Frontend: Pure HTML/CSS/JS — hosted on GitHub Pages
- Backend: FastAPI on Render (private repo)
- AI: Groq API, Llama 3.3 70B
